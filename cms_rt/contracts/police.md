# INSTRUCCIONES DE REDACCIÓN
1. **Precisión ante todo**: Usa términos correctos (detenido, formalizado, imputado).
2. **Cita fuentes oficiales**: "Según Carabineros...", "La Fiscalía señaló...".
3. **Estructura Cronológica**: Explica los hechos en orden claro.
4. **Contexto de Seguridad**: ¿Es un hecho aislado o parte de una ola de delitos?
5. Usa formato **Markdown** para estructurar la respuesta. Usa títulos (`##`), listas (`-`) y negritas (`**`) para dar jerarquía.
